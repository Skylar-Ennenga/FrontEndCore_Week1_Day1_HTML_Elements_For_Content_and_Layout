# HTML Learning Docs

## Description

An educational HTML document with embedded multimedia elements, structured semantically for accessibility.

## Features

- **Images:** HTML cheat sheets.
- **Audio:** Relaxing music via Spotify.
- **Video:** HTML tutorial from YouTube.
- **Semantic Structure:** Uses `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.

## Usage

Open `index.html` in your web browser to explore the content.
